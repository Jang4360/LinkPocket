# reference — 외부 자료 & 프로젝트 목적

소프트웨어 문서가 아니라, **판단의 근거가 된 외부 자료**와 **이 프로젝트의 목적(취업) 서사**를 둔다.

| 파일 | 무엇 |
|---|---|
| [sources.md](sources.md) | 기술 블로그·공식 문서·채용 공고·법령 등 조사 근거 링크 |
| [career-narratives.md](career-narratives.md) | 채용 신호, 남길 최소 산출물, 회사 유형별 서사. 발행한 블로그·면접 답변 최종본도 여기서 관리. |

> `career-narratives.md`는 원래 `career/` 폴더에 있었으나, **취업은 이 프로젝트를 하는 목적이지 소프트웨어 자체의 문서가 아니므로** reference로 옮겼다. 개발 문서 폴더(product·decisions·plan·architecture·operations…)와 성격이 다르다.
